Privacy - Delivery Address Extractor

This privacy page applies only to Delivery Address Extractor.

  • The app processes user-provided delivery message text.
  • The user or calling system provides the message text.
  • The data is used for extracting structured delivery fields for the current request.
  • No login required.
  • No user account required.
  • No long-term storage of submitted input by this app.
  • No external API access by this app.
  • The app does not contact couriers.
  • The app does not modify orders.
  • No sale of user data.
  • No use for advertising.
  • No background processing.
