Support Message Classifier

Classify customer support messages and extract fixed structured fields for routing, ticket triage, or human review before support handling.

Privacy Terms Support

What It Does

This app classifies customer support messages, extracts fixed structured fields, and returns structured JSON for support routing, ticket triage, or human review.

Returned Fields

  • issue_type
  • user_request
  • urgency_label
  • missing_fields
  • source_text
  • errors

Boundaries

  • It does not reply to customers.
  • It does not promise a solution.
  • It does not update ticket systems.
  • It does not contact customers.
  • It does not call external APIs.
  • It does not make subjective support recommendations.
  • It does not modify any external records.
