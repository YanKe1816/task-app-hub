Privacy - Refund Request Extractor

This privacy page applies only to Refund Request Extractor.

  • The app processes customer refund message text submitted by the user.
  • The submitted text is used only to return structured extraction results for the current request.
  • No login required.
  • No user account required.
  • No long-term storage of submitted input.
  • No database used for submitted input.
  • No external API access.
  • No customer contact.
  • No order modification.
  • No sale of user data.
  • No use of submitted content for advertising.

Users should avoid submitting unnecessary sensitive personal information.
