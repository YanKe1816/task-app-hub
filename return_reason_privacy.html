Privacy - Return Reason Normalizer

This privacy page applies only to Return Reason Normalizer.

  • The app processes only the return reason text submitted by the user.
  • The submitted text is used only to return the structured normalization result for the current request.
  • No login is required.
  • No user account is required.
  • No long-term storage of submitted input.
  • No external API access.
  • No sale of user data.
  • No use for advertising.
  • The app does not contact customers.
  • The app does not modify orders.

Users may contact sidcraigau@gmail.com for privacy or data-related requests.
